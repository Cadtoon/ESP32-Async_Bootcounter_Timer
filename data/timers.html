Before you get started here, these timer require the NTP Server to be functioning. The NTP Server requires the internet, so before you do anything, configure the network, restart the ESP32, then open this page through the local network IP.

Here there is two timers. Timer1 is programmed to turn the ESP32s built-in LED (GPIO 02) on and off, providing your ESP32 has one. My Wroom has a red power LED and a second blue LED on GPIO 02. Timer2 has no programmimg behind it. If your connected to a computer useing the USB, the Serial Monitor does provide messages for both timers On and Off activations.

Using the controls for Timer1, select a time in the "Timer1 On" row. Pick a time that is within a few minutes from the current time. Then click the Update button. Now select a different time for the "Timer1 Off", again withing a few minutes and click the Update button. The "Timer1 Switch" is used to turn the timer On or Off. By default it is in the Off position. Move the Timer1 slider to the On position, click the Update button. If you select the exact time for both "Timer1 On" and "Timer1 Off", it will disable the timer, The same as having the "Timer1 Switch" in the Off position. Now sit back and watch the LED.
